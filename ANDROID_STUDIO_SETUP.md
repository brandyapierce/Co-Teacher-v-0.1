## Step 2: Run the Installer (5-10 minutes)

1. **Double-click**: The downloaded `android-studio-*.exe` file
2. **Click**: "Yes" if User Account Control asks for permission
3. **Welcome Screen**: Click "Next"
4. **Choose Components**: 
   - ‚úÖ Keep "Android Studio" checked
   - ‚úÖ Keep "Android Virtual Device" checked (optional but recommended)
   - Click "Next"
5. **Installation Location**: 
   - Default is fine: `C:\Program Files\Android\Android Studio`
   - Click "Next"
6. **Start Menu Folder**: 
   - Default is fine
   - Click "Install"
7. **Wait**: For installation to complete (~5 minutes)
8. **Click**: "Next" then "Finish"

## ‚ö†Ô∏è Troubleshooting

### Download is slow?
- Normal for first-time setup (2-3 GB of tools)
- Can take 15-30 minutes depending on internet speed

### Installation failed?
- Make sure you have ~5-6 GB free disk space
- Run installer as Administrator
- Disable antivirus temporarily if it's blocking

### License acceptance issues?
- Make sure to click "Accept" for ALL licenses
- Some licenses require scrolling to the bottom first
