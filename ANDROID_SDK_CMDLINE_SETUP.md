**Issue**: Command-line tools are missing from Android SDK  
**Solution**: Install them via Android Studio  
**Time**: 5 minutes

---

## Step 1: Open Android Studio

1. **Press Windows key** and type "Android Studio"
2. **Click** "Android Studio" to launch it
3. **Wait** for it to load (may take 30 seconds)

---

## Step 2: Open SDK Manager

### If you see the Welcome screen:
1. **Click** "More Actions" (three dots button)
2. **Select** "SDK Manager"

### If you see a project open:
1. **Click** "File" menu â†’ "Settings"
2. **Navigate** to: Appearance & Behavior â†’ System Settings â†’ **Android SDK**

---

## Step 3: Install Command-Line Tools

1. **Click** the "SDK Tools" tab (top of window)
2. **Check** the box next to "Android SDK Command-line Tools (latest)"
3. **Click** "Apply" button at the bottom
4. **Click** "OK" on the confirmation dialog
5. **Wait** for download to complete (~100 MB, 2-3 minutes)
6. **Click** "Finish" when done
7. **Click** "OK" to close SDK Manager

---

## Step 4: Close Android Studio

You can close Android Studio now - we don't need it open anymore!

---

**Ready?** Follow the steps above, then type "done" when finished!
